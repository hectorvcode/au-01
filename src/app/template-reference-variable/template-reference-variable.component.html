Template Reference Variable

You can also use the template reference variable to create a template variable #name to an input box, and you can use it get the value of the input field
{ { name.value } }

Welcome {{ name.value }}

We also use (keyup)='0' on the input element. Angular updates the view, when it runs the change detection. The change detection runs only in response to asynchronous events.

A template reference variable is ofter a reference to a DOM element within a template. You can easily access the variable anywhere in the template. You declare it by using the hash symbol(#)
